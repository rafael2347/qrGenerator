# Generador de QR

¡Bienvenido al Generador de QR! Esta es una aplicación web construida con **React** y **Vite** que permite a los usuarios crear códigos QR de manera rápida y sencilla.

## Características

- Generación de códigos QR a partir de texto, URLs, o cualquier dato que desees codificar.
- Interfaz de usuario intuitiva y fácil de usar.
- Opción para descargar el código QR generado.
- Compatible con dispositivos móviles y de escritorio.

## Tecnologías Utilizadas

- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **Vite**: Herramienta de construcción rápida y ligera para aplicaciones modernas.
- **QRCode.react**: Librería para la generación de códigos QR en React.

## Instalación

Para comenzar a usar la aplicación, sigue estos pasos:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tu_usuario/generador-qr.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd generador-qr
   ```

3. Instala las dependencias:

   ```bash
   npm install
   ```

4. Inicia el servidor de desarrollo:

   ```bash
   npm run dev
   ```

5. Abre tu navegador y visita `http://localhost:3000` para ver la aplicación en acción.

## Uso

1. Ingresa el texto o URL que deseas convertir en un código QR.
2. Haz clic en el botón "Generar QR".
3. Descarga el código QR si lo deseas.
